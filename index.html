Interactive Fiction with Ink

Welcome! This page guides you through creating interactive stories in Ink. Follow the steps and experiment with the code.

Step 1: Install Inky

Download Inky to run your Ink stories.

Step 2: Basic Story

This is your first story with crossroads and choices:


=== start ===
You are at a crossroads.

+ [Go left] -> left
+ [Go right] -> right

=== left ===
You find a treasure chest!
You win!
+ [Play again] -> start

=== right ===
You fall into a trap!
Game over.
+ [Try again] -> start

Copy this code into Inky and play it. Notice the === knots === and + choices.

Step 3: Variables and Inventory

Here’s a version with lives and a key:


VAR has_key = false
VAR lives = 3
VAR previous_knot = "start"

=== start ===
{lives > 0:
    What do you do at the crossroads?
    + [Go left] -> left
    + [Go right] -> right
    + [Check your backpack] -> check_inventory
- else:
    You’re out of lives! Game over.
    + [Try again?] -> restart_game
}

=== left ===
~ previous_knot = "left"
You find a shiny key!
{has_key == false:
    ~ has_key = true
    You pick it up and put it in your backpack.
- else:
    You already have the key.
}
+ [Go back] -> start

=== right ===
~ previous_knot = "right"
You fall into a trap!
~ lives = lives - 1
{lives > 0:
    You lose a life. You have {lives} lives left.
    -> start
- else:
    You’re out of lives! Game over.
    + [Try again?] -> restart_game
}

=== check_inventory ===
{has_key:
    You have a key in your backpack.
- else:
    Your backpack is empty.
}
+ [Return] -> return_from_inventory

=== return_from_inventory ===
-> {previous_knot}

=== restart_game ===
~ lives = 3
~ has_key = false
-> start

Step 4: Play, Modify, Experiment

Try these exercises:

  • Add a new knot with a hidden treasure.
  • Make the right path lose 2 lives instead of 1.
  • Add another item in the backpack.
