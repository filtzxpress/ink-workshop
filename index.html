2. Example: A Simple Game with Lives and Items

VAR lives = 3
VAR previous_knot = "start"

+ [Start the game] -> start

=== start ===
{lives > 0:
    You are at a crossroads. What do you do?

    + [Go left] -> left_path

    ~ has_key = true
