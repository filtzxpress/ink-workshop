Interactive Fiction with Ink – Beginner Friendly

Welcome! This page guides you through creating interactive stories using Ink. We’ll break down the code, explain variables, conditionals, knots, diverts, and how the computer reads and executes your story. Don’t worry if it seems confusing at first — step by step, it will make sense.

Step 1: Install Inky

Download Inky and open it on your computer. You can copy code from this page, paste it into Inky, and run it immediately.

Step 2: Basic Story – Knots, Choices, Diverts

Knots: Named sections of your story, like chapters. The computer jumps between knots using diverts.
Choices: Player decisions. "+" allows multiple selections; "*" is one-time only.
Diverts (->): Tells the computer where to go next. Without a divert, the story ends.
Computer perspective: The computer reads the story line by line. When it sees a choice, it waits for input. When a choice is selected, it jumps to the specified knot and continues. 
    
    
=== start ===   // Knot starts here
You are at a crossroads.

+ [Go left] -> left     // Choice → jump to 'left'
+ [Go right] -> right   // Choice → jump to 'right'

=== left ===
You find a treasure chest!
You win!
+ [Play again] -> start  // Choice → restart story

=== right ===
You fall into a trap!
Game over.
+ [Try again] -> start    // Choice → restart story

Step 3: Variables, Conditionals, Inventory

Variables: The computer uses them to "remember" state. - Boolean: True/False (e.g., has_key = false). - Integer: Whole numbers (e.g., lives = 3). Conditionals: Only run code if a condition is true. Diverts in conditionals: The computer jumps to different knots depending on variable values. 
    
    
VAR has_key = false        // Boolean: tracks if player has key
VAR lives = 3              // Integer: tracks remaining lives
VAR previous_knot = "start" // Track previous knot

=== start ===
{lives > 0:                // Conditional: if lives > 0
    What do you do at the crossroads?
    + [Go left] -> left
    + [Go right] -> right
    + [Check your backpack] -> check_inventory
- else:
    You’re out of lives! Game over.
    + [Try again?] -> restart_game
